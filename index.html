Password Checker and Report Generator

Enter a value or upload a CSV file (username,password). The browser computes SHA-1, sends only the first 5 hex chars to https://api.pwnedpasswords.com/range/, and checks locally. After a run you can generate a detailed report (JSON/HTML) containing indicators and recommendations.

Privacy notes: SHA-1 computed locally (Web Crypto). Use a trusted password manager and change passwords immediately if they appear in breaches.
